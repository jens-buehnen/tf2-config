***HOW TO INSTALL***

Place the vpk file into your ".../Team Fortress 2/tf/custom" folder
When you boot up TF2 just create a server (load up the itemtest map for quicker load time)
or type "map itemtest" into the console.

OR 

Place the vpk file into your ".../Team Fortress 2/tf/custom" folder
Inside ".../Team Fortress 2/tf/cfg" open "autoexec.cfg" (If you don't have one then make one)
In "autoexec.cfg" paste this line "map itemtest; wait 5; disconnect; wait 3; echo" (without quotes)
Everytime you boot up TF2 it will load up itemtest for you then unload the map so you don't have to do anything.

If you've done everything correct then the animations should be working and now you can enter any Valve or community server you like (assuming they have sv_pure turned off)

Enjoy!
